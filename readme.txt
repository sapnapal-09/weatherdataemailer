# Weather Data Emailer

## Description
A PHP-based web application to fetch weather data, summarize it using OpenAI, and email it to a recipient.



## Usage
- Access the form at `http://localhost:8000/index.html`.
- Enter location and recipient email.
- Check the recipient email for the weather report.
